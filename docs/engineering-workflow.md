# Engineering Workflow

## Tier model

```
Tier 1: Local (validate:fast)  →  pre-push hook (git)
Tier 2: PR gates               →  pr-validation.yml (10 jobs)
Tier 3: Nightly                →  governance / e2e / performance
```

### Tier 1 — local (< 60s)

```bash
npm run validate:fast
# Runs: tsc --noEmit → typecheck:services → validate:architecture → vitest (contracts + invariants)
```

Installed as a pre-push hook via:
```bash
git init          # if not already
npm run hooks:install
```

### Tier 2 — PR gates

`.github/workflows/pr-validation.yml` — 10 jobs:
- typecheck (frontend + backend)
- unit tests (vitest)
- invariant tests
- contract tests
- architecture (dependency-cruiser)
- migration-safety (pure git diff)
- openapi-diff
- blast-radius
- security-audit (npm audit)
- build (vite)

All pass before merge.

### Tier 3 — nightly

- `nightly-governance.yml` — full Semgrep, depcheck, ts-prune
- `e2e.yml` — Playwright
- `performance.yml` — Lighthouse + k6

## Command reference

### Validate

| Command | What it does | Approx time |
|---|---|---|
| `npm run validate` | Full suite (all checks) | 2-5 min |
| `npm run validate:fast` | tsc + typecheck:services + dep-cruiser + vitest (contracts/invariants) | < 60s |
| `npm run validate:contracts` | Vitest contract tests only | 15s |
| `npm run validate:architecture` | dependency-cruiser boundary rules | 10s |
| `npm run validate:semgrep` | Semgrep (CI-only on Windows — no native build) | 30s |
| `npm run validate:performance` | Lighthouse CI | 2 min |
| `npm run validate:memory` | MemLab | 3 min |

### Report

| Command | What it does |
|---|---|
| `npm run report:deadcode` | `ts-prune` — find unused exports |
| `npm run report:deps` | `depcheck` — find unused/missing deps |

### CI local

```bash
npm run ci:local
# Runs `act` — simulates PR validation locally (requires Docker)
```

### Hooks

```bash
npm run hooks:install
# Creates .git/hooks/pre-push running validate:fast
```
